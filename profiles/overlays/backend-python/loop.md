## Backend / Python (FastAPI) loop extension

Расширение dev-pipeline для Python-бекенда.

### Trigger phrases
- EN: `next backend task`, `add endpoint`, `run migrations`, `test with coverage`, `run api`
- RU: `следующая задача`, `добавь эндпоинт`, `прогони миграции`, `тесты с покрытием`, `запусти api`

### Pipeline
Стандартный dev-pipeline. architect/implementer/tester/refactor-agent/bug-hunter/reviewer знают:
- FastAPI + Starlette (async endpoints, dependency injection, background tasks)
- SQLAlchemy 2.x async (async_sessionmaker, AsyncSession, `select()` statements)
- Pydantic 2 (BaseModel, `model_config`, `Field`, validators)
- Alembic (autogenerate + manual review)
- uv (mandatory package manager for new projects)
- pytest + pytest-asyncio + httpx AsyncClient + factory-boy или model_bakery
- ruff (linter + formatter) + mypy (strict)
- asyncio best practices (structured concurrency via `asyncio.TaskGroup`, `anyio` for portability)

### Специальные диспатчи
| Задача | Агент |
|---|---|
| Пакеты (install / lock / sync) | `uv-manager` |
| Тесты | `pytest-runner` |
| Миграции (autogenerate / upgrade / downgrade) | `alembic-manager` |
| Стиль + форматирование | `ruff-checker` |
| Типы | `mypy-checker` |
| Первичный scaffold FastAPI | `init-fastapi` |

### Изоляция — специфичные правила
- **`uv.lock` может быть 5000+ строк** — использовать `uv tree` или `grep` вместо чтения целиком.
- **Alembic autogenerated migrations ВСЕГДА требуют человеческого ревью** — implementer никогда не коммитит `alembic revision --autogenerate` без diff'а.
- **`__pycache__/` и `.venv/` НИКОГДА не читаем** — только исходники.
- **Pytest output может быть 5k+ строк** при failure cascade — `pytest-runner` возвращает только first failure + last N + summary.
- **Не запускать `alembic upgrade head` на prod-БД без подтверждения**. Dev-БД — можно.
- **Не устанавливать пакеты через `pip install`** — только `uv add` или `uv sync`. Смешивание ломает lockfile.
- **Никогда не коммитить `.env`** — только `.env.example`.
- **Никогда не логгировать `settings.database_url`, `settings.jwt_secret`, `settings.openai_api_key`** — эти поля должны быть `SecretStr` в Pydantic Settings.
